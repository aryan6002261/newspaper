# 📰 News Weekly - Newspaper Layout Project

A simple and elegant newspaper-style web page built using **HTML** and **CSS**. This project recreates the classic newspaper layout with a featured headline section and a multi-column news grid.

## 📌 Features

- Newspaper-inspired design
- Featured headline section
- Multi-column article layout using CSS Grid
- Clean typography and structured content
- Responsive and lightweight implementation
- Pure HTML and CSS (no frameworks or libraries)

## 🚀 Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```text
newspaper-project/
│
├── newspaper.html
├── styles.css
└── README.md
```

## 🖥️ Preview

The page contains:

- A newspaper header displaying:
  - Publication date
  - Newspaper title (**News Weekly**)
  - Category label (**Global News**)

- A featured headline article covering a breakthrough in solar cell technology.

- Four news sections:
  - Local News
  - International Affairs
  - City News
  - Business News

## 🔧 How to Run

1. Clone the repository:

```bash
git clone https://github.com/aryan6002261/newspaper.git
```

2. Navigate to the project folder:

```bash
cd newspaper
```

3. Open `newspaper.html` in your preferred web browser.

No additional setup or dependencies are required.
